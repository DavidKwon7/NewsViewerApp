# NewsViewerApp
뉴스 관련 open API를 사용하여, 뉴스 기사 검색 및 저장&불러오기&삭제 기능 구현 

2022/11/7 프로젝트 시작 ~ 현재 진행 중.. 

## 사용 기술 
- kotlin
- kotlin DSL & buildSrc
- multi module
- coroutine
- flow 
- activity & fragment
- navigation 
- hilt
- retrofit
- room
- paging 3
- glide
